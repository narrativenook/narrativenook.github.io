7. Llama 3 70B on a Single 4GB GPU

Meet Llama 3 70B—a super-efficient computer that uses "layer-wise inference" to tackle complex tasks while saving loads of space and making things super speedy. What's more, this model can run on a single 4GB GPU, making it a remarkable achievement in AI innovation.



8. Unlocking the Power of Function Calling

The 70B parameter version of Llama-3 takes a giant leap forward with its function calling capability, elevating it beyond simple text generation. This groundbreaking feature enables Llama-3 to interact with external data sources, analyze information, and generate actionable insights.

For instance, you can create a function that retrieves customer data from a database and feeds it into Llama-3. The model can then analyze the data and generate personalized marketing emails, demonstrating its potential to revolutionize various industries.

10. Private LLM on iPhone with Llama-3-Smaug

Developed by Abacus.AI, Llama-3-Smaug is a fine-tuned version of the powerful Meta Llama-3. This fine-tuning focuses on creating engaging, multi-turn dialogues through techniques like Direct Preference Optimisation (DPO) and DPO-Positive (DPOP).

The Private LLM guys made Llama-3-Smaug from Abacus available on their iPhone app.
